# rc

Arquivos úteis de configuração/execução.

## Instalação

+ Clone o repositório e execute o script `install.sh`:

        git clone https://github.com/flaudisio/rc.git ~/.rc
        cd ~/.rc
        ./install.sh

+ Use `./install.sh -h` para as opções de instalação.
