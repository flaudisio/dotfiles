# rc

Arquivos úteis de configuração/execução.

## Instalação

Basta executar o script `install.sh`:

    ~ $ ./install.sh -h
    Uso: ./install.sh [-f] [-P] [-h]

    Opções:
        -f, --force         Sobrescreve os .arquivos existentes.
        -P, --no-pathogen   Não instala o pathogen.vim.
        -h, --help          Esta mensagem.
